## 🎨 UI 组件 (shared/components/)

### `GlassCard.vue` - 玻璃卡片组件

**作用**: 提供毛玻璃效果的卡片容器

**功能**:
- 半透明背景
- 模糊效果（backdrop-filter）
- 边框和阴影
- 支持点击、长按等交互

**使用示例**:
```vue
<GlassCard @click="handleClick">
  <div class="content">
    <!-- 卡片内容 -->
  </div>
</GlassCard>
```

**样式特点**:
- 背景: 半透明白色渐变
- 模糊: 30px 背景模糊
- 边框: 2px 白色半透明边框
- 阴影: 多层阴影效果
- 尺寸: 200x200px（可自定义）

**应用场景**:
- 标准模式页面的设备卡片
- 任何需要毛玻璃效果的容器

### `NavBar.vue` - 导航栏组件

**作用**: 房间标签切换和管理

**功能**:
1. **显示房间标签**
   - 横向滚动显示所有房间
   - 当前激活的房间高亮显示

2. **切换房间**
   - 点击标签切换到对应房间
   - 自动滚动到激活的标签

3. **添加房间**
   - 点击"+"按钮打开添加对话框
   - 输入房间名称
   - 检查重复
   - 添加新房间

4. **删除房间**
   - 右键点击标签打开删除确认
   - 确认后删除房间
   - 删除房间时同时删除该房间的所有设备

**使用示例**:
```vue
<NavBar />
```

**交互方式**:
- 左键点击: 切换房间
- 右键点击: 删除房间
- 点击"+": 添加房间
- 拖动: 滚动查看更多房间

**状态管理**:
- 使用 `useTabsStore()` 管理房间数据
- 房间数据持久化到 localStorage

**样式特点**:
- 毛玻璃效果背景
- 激活标签有特殊样式
- 平滑滚动动画
- 响应式设计

### `Clock.vue` - 时钟组件

**作用**: 显示当前日期和时间

**功能**:
- 显示星期几
- 显示当前时间（时:分）
- 实时更新（每秒）

**使用示例**:
```vue
<Clock />
```

**显示格式**:
```
星期一
14:30
```

**实现原理**:
```typescript
const updateTime = () => {
  const now = new Date()
  const days = ['星期日', '星期一', '星期二', '星期三', '星期四', '星期五', '星期六']
  
  date.value = days[now.getDay()]
  time.value = `${String(now.getHours()).padStart(2, '0')}:${String(now.getMinutes()).padStart(2, '0')}`
}

// 每秒更新
setInterval(updateTime, 1000)
```

**样式特点**:
- 日期: 15px，半透明
- 时间: 48px，加粗
- 文字阴影效果

### `BrightnessControl.vue` - 亮度控制组件

**作用**: 控制灯光设备的亮度

**功能**:
1. **显示设备信息**
   - 设备名称

2. **亮度调节**

**显示条件**:

## 🧩 共享逻辑 (shared/composables/)

### `useNameOverflow.ts` - 文本溢出检测

**作用**: 检测元素的文本是否超出容器宽度

**功能**:
- 检查文本是否溢出
- 返回溢出状态
- 用于决定是否显示滚动动画

**使用示例**:
```typescript
import { useNameOverflow } from '@/shared/composables'

const { checkNameOverflow, isNameOverflow } = useNameOverflow()

// 检查元素是否溢出
checkNameOverflow(element, deviceId)

// 判断是否溢出
if (isNameOverflow(deviceId)) {
  // 添加滚动动画类
}
```

**在模板中使用**:
```vue
<template>
  <div 
    class="device-name" 
    :ref="(el) => checkNameOverflow(el as HTMLElement, device.id)"
  >
    <span 
      class="name-text" 
      :class="{ overflow: isNameOverflow(device.id) }"
    >
      {{ device.name }}
    </span>
  </div>
</template>

<style>
.name-text.overflow {
  animation: scroll 3s linear infinite;
}

@keyframes scroll {
  0%, 20% { transform: translateX(0); }
  80%, 100% { transform: translateX(-50%); }
}
</style>
```

**实现原理**:
```typescript
const overflowMap = ref<Record<string, boolean>>({})

const checkNameOverflow = (el: HTMLElement, id: string) => {
  if (!el) return
  
  nextTick(() => {
    const isOverflow = el.scrollWidth > el.clientWidth
    overflowMap.value[id] = isOverflow
  })
}

const isNameOverflow = (id: string) => {
  return overflowMap.value[id] || false
}
```

**应用场景**:
- 设备名称过长时显示滚动动画
- 任何需要检测文本溢出的地方

## 🛠️ 共享工具 (shared/utils/)

### `timeFormat.ts` - 时间格式化工具

**作用**: 格式化时间显示

#### `formatRemaining()` - 格式化剩余时间

**功能**: 将秒数转换为易读的时间格式

**使用示例**:
```typescript
import { formatRemaining } from '@/shared/utils'

formatRemaining(3665)  // "1小时1分"
formatRemaining(125)   // "2分5秒"
formatRemaining(45)    // "45秒"
formatRemaining(3600)  // "1小时"
formatRemaining(60)    // "1分"
```

**格式规则**:
- 大于1小时: 显示"X小时X分"
- 大于1分钟: 显示"X分X秒"
- 小于1分钟: 显示"X秒"
- 整小时: 只显示"X小时"
- 整分钟: 只显示"X分"

**实现代码**:
```typescript
export function formatRemaining(seconds: number): string {
  const hours = Math.floor(seconds / 3600)
  const minutes = Math.floor((seconds % 3600) / 60)
  const secs = seconds % 60
  
  if (hours > 0) {
    return minutes > 0 
      ? `${hours}小时${minutes}分` 
      : `${hours}小时`
  }
  
  if (minutes > 0) {
    return secs > 0 
      ? `${minutes}分${secs}秒` 
      : `${minutes}分`
  }
  
  return `${secs}秒`
}
```

**应用场景**:
- 显示定时器倒计时
- 显示倒计时剩余时间
- 任何需要格式化时间的地方

## 📚 布局功能 (features/layout/)

### `tabs.ts` - 房间标签管理

**作用**: 使用 Pinia 管理房间标签数据

**位置**: `src/features/layout/components/tabs.ts`

#### 状态 (State)

```typescript
{
  tabs: string[],        // 房间列表
  activeIndex: number    // 当前激活的房间索引
}
```

#### 方法 (Actions)

```typescript
// 添加房间
addTab(name: string): void

// 删除房间
removeTab(index: number): void

// 设置激活的房间
setActiveIndex(index: number): void
```

#### 使用示例

```typescript
import { useTabsStore } from '@/features/layout/components/tabs'

const store = useTabsStore()

// 添加房间
store.addTab('客厅')

// 删除房间
store.removeTab(0)

// 切换房间
store.setActiveIndex(1)

// 获取当前房间
const currentRoom = store.tabs[store.activeIndex]

// 获取所有房间
const allRooms = store.tabs
```

#### 数据持久化

- 使用 `localStorage` 存储房间数据
- 应用启动时自动加载
- 数据变化时自动保存

#### 初始数据

如果 localStorage 中没有数据，会创建默认房间：
```typescript
['客厅', '卧室', '厨房']
```

## 💡 使用建议

### 添加新的共享组件
1. 在 `shared/components/` 创建组件
2. 确保组件是通用的，不包含业务逻辑
3. 在需要的地方导入使用

### 添加新的 Composable
1. 在 `shared/composables/` 创建文件
2. 导出可复用的逻辑函数
3. 在 `index.ts` 中统一导出

### 添加新的工具函数
1. 在 `shared/utils/` 创建文件
2. 导出纯函数（无副作用）
3. 在 `index.ts` 中统一导出

### 添加新的图标
1. 打开 `shared/components/SvgIcon.vue`
2. 在 `icons` 对象中添加新图标的 SVG 代码
3. 使用 `<SvgIcon name="新图标名" />`
