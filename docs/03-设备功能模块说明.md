# 设备功能模块详细说明

设备功能模块位于 `src/features/device/`，包含所有与设备相关的代码。

---

## 📁 目录结构

```
features/device/
├── components/      # 设备组件
├── composables/     # 设备逻辑
├── utils/           # 设备工具
└── stores/          # 设备数据
```

---

## 🎨 设备组件 (components/)

### 底部控制栏组件

#### `BottomPowerBar.vue` - 通用电源控制栏

**作用**: 为不需要特殊控制的设备提供简单的电源开关

**适用设备**: 开关、路由器等

**功能**:
- 显示设备名称和类型
- 提供电源开关按钮
- 从底部滑入动画

**使用示例**:
```vue
<BottomPowerBar 
  :visible="showBottomPowerBtn" 
  :device="bottomPowerDevice" 
  @click="onBottomPowerClick" 
/>
```

**事件**:
- `click`: 点击电源按钮

```vue

  @lift-down="handleLiftDown"
```

### 对话框组件

#### `DeviceDialog.vue` - 设备详情对话框

**作用**: 提供完整的设备控制功能（最复杂的组件）

**适用设备**: 除插座外的所有设备

**功能**:

1. **通用功能**
   - 显示设备名称、类型、状态
   - 提供电源开关
   - 提供定时器设置
   - 提供重启功能（路由器）

2. **灯光设备专用**
   - 亮度调节滑块
   - 色温调节滑块
   - 模式选择（阅读、电脑、休闲、办公、娱乐、夜灯）

3. **扫地机器人专用**
   - 清扫模式选择（扫地、拖地、扫拖同时、先扫后拖）
   - 吸力调节（一档、二档、三档）- 支持滑动和滚轮
   - 水量调节（少、中、多）- 支持滑动和滚轮
   - 回充按钮

4. **晾衣架专用**
   - 升降控制（上升、停止、下降）
   - 功能开关（照明、烘干、风干、消毒）

5. **洗衣机专用**
   - 模式选择（标准洗、快速洗、强力洗等）
   - 开始清洗按钮

**使用示例**:
```vue
<DeviceDialog
  v-model:visible="showDeviceDialog"
  :device="selectedDevice"
  @toggle-power="handleTogglePower"
  @open-timer="handleOpenTimer"
  @restart="handleRestart"
  @update:brightness="handleBrightnessUpdate"
  @update:color-temp="handleColorTempUpdate"
/>
```

**Props**:
- `visible`: 是否显示对话框
- `device`: 设备对象

**事件**:
- `update:visible`: 更新显示状态
- `toggle-power`: 切换电源
- `open-timer`: 打开定时器设置
- `restart`: 重启设备
- `update:brightness`: 更新亮度
- `update:color-temp`: 更新色温

**核心逻辑**:

1. **根据设备类型显示不同界面**
```typescript
const getControlButtons = (device: Device) => {
  switch (device.type) {
    case 'light':
      return [电源, 定时]
    case 'cleaner':
      if (device.cleanerType === 'robot-vacuum') {
        return [清扫/暂停, 回充]
      }
      // ... 其他清洁电器
    // ... 其他类型
  }
}
```

2. **滑动控制吸力和水量**
```typescript
const suctionSlider = useVerticalSlider(adjustSuction)
const waterSlider = useVerticalSlider(adjustWater)

// 在模板中使用
@mousedown="suctionSlider.onDragStart"
@mousemove="suctionSlider.onDragMove"
@mouseup="suctionSlider.onDragEnd"
@wheel="suctionSlider.onWheel"
```

3. **模式选择**
```typescript
const handleModeSelect = (mode: any, index: number) => {
  setModeIndex(device.id, index)
  
  // 如果是灯光模式，同时设置亮度和色温
  if (typeof mode === 'object' && mode.brightness) {
    emit('update:brightness', mode.brightness)
    emit('update:colorTemp', mode.colorTemp)
  }
}
```

#### `TimerDialog.vue` - 定时器设置对话框

**作用**: 设置设备的定时开启和定时关闭

**功能**:
- 设置定时关闭时间
- 设置定时开启时间
- 设置是否重复执行
- 支持拖动调节时间
- 显示当前定时器状态
- 取消所有定时器

**使用示例**:
```vue
<TimerDialog
  v-model:visible="showTimerDialog"
  :device="selectedDevice"
  @confirm="handleTimerConfirm"
  @cancel-all="handleTimerCancelAll"
/>
```

**事件**:
- `confirm`: 确认设置，返回定时器数据
- `cancel-all`: 取消所有定时器

**数据格式**:
```typescript
{
  offEnabled: boolean,      // 是否启用定时关闭
  onEnabled: boolean,       // 是否启用定时开启
  offTime: string,          // 关闭时间 "22:00"
  onTime: string,           // 开启时间 "07:00"
  offRepeat: boolean,       // 关闭是否重复
  onRepeat: boolean         // 开启是否重复
}
```

**交互方式**:
- 点击时间显示区域开始拖动
- 上下拖动调节时间
- 支持小时和分钟分别调节

## 🧩 设备逻辑 (composables/)

### `useDeviceControl.ts` - 设备控制状态管理

**作用**: 提供设备控制相关的可复用逻辑

#### 1. `useDeviceState()` - 设备状态管理

**功能**:
- 管理设备的模式索引
- 管理控制按钮索引
- 管理洗衣机清洗状态
- 判断是否可以开始清洗

**使用示例**:
```typescript
const {
  activeModeIndex,      // 当前激活的模式索引
  activeControlIndex,   // 当前激活的控制索引
  isWashingActive,      // 是否正在清洗
  canStartWash,         // 是否可以开始清洗
  setModeIndex,         // 设置模式索引
  setControlIndex,      // 设置控制索引
  toggleWashing         // 切换清洗状态
} = useDeviceState(deviceRef)
```

**应用场景**:
- 在 DeviceDialog 中管理设备的各种状态
- 每个设备独立维护自己的状态

#### 2. `useRobotControl()` - 扫地机器人控制

**功能**:
- 管理吸力档位（0-2）
- 管理水量档位（0-2）
- 提供调整函数

**使用示例**:
```typescript
const {
  suctionLevel,      // 吸力档位 0-2
  waterLevel,        // 水量档位 0-2
  suctionLevels,     // ['一档', '二档', '三档']
  waterLevels,       // ['少', '中', '多']
  adjustSuction,     // 调整吸力 adjustSuction(1) 或 adjustSuction(-1)
  adjustWater        // 调整水量
} = useRobotControl()
```

**应用场景**:
- 在 DeviceDialog 中控制扫地机器人的吸力和水量

#### 3. `useClothesRackControl()` - 晾衣架控制

**功能**:
- 管理照明开关状态
- 管理烘干开关状态
- 管理风干开关状态
- 管理消毒开关状态
- 管理升降动作状态

**使用示例**:
```typescript
const {
  lighting,      // 照明开关
  drying,        // 烘干开关
  airDry,        // 风干开关
  sterilize,     // 消毒开关
  moving,        // 升降动作 'up' | 'down' | 'stop' | null
  reset          // 重置所有状态
} = useClothesRackControl()
```

**应用场景**:
- 在 DeviceDialog 中控制晾衣架的各种功能

### `useSliderControl.ts` - 滑动控制

**作用**: 提供滑动和滚轮控制的通用逻辑

#### 1. `useVerticalSlider()` - 垂直滑动控制

**功能**:
- 处理触摸和鼠标拖动事件
- 处理滚轮事件
- 自动计算滑动方向（向上/向下）
- 达到阈值时触发回调

**使用示例**:
```typescript
const slider = useVerticalSlider((delta) => {
  // delta 为 1（向上）或 -1（向下）
  adjustSuction(delta)
})

// 在模板中使用
<div
  @mousedown="slider.onDragStart"
  @mousemove="slider.onDragMove"
  @mouseup="slider.onDragEnd"
  @wheel="slider.onWheel"
>
  <!-- 滑动区域 -->
</div>
```

**配置**:
- 拖动阈值: 40px（在 `uiConfig.ts` 中配置）
- 滚轮阈值: 10px

#### 2. `useHorizontalSlider()` - 水平滑动控制

**功能**: 与垂直滑动类似，但方向为水平

**使用场景**:
- 水平滑动调节参数
- 目前项目中未使用，预留接口

## 💾 设备数据 (stores/)

### `devices.ts` - 设备状态管理

**作用**: 使用 Pinia 管理所有设备数据

#### 状态 (State)

```typescript
{
  devices: Device[]  // 所有设备列表
}
```

#### 方法 (Actions)

##### 设备管理
```typescript
// 添加设备
addDevice(device: Omit<Device, 'id'>): void

// 删除设备
removeDevice(id: string): void

// 切换设备状态（开启/关闭）
toggleStatus(id: string): void
```

##### 灯光控制
```typescript
// 设置亮度
setBrightness(id: string, brightness: number): void

// 设置色温
setColorTemp(id: string, colorTemp: number): void
```

##### 定时器管理
```typescript
// 设置定时器
setTimer(
  id: string,
  type: 'on' | 'off',
  enabled: boolean,
  time: string,
  repeat: boolean
): void

// 取消定时器
cancelTimer(id: string, type: 'on' | 'off'): void

// 获取定时器剩余时间（秒）
getTimerRemaining(id: string, type: 'on' | 'off'): number
```

##### 倒计时管理
```typescript
// 设置倒计时
setCountdown(
  id: string,
  action: 'on' | 'off',
  seconds: number
): void

// 取消倒计时
cancelCountdown(id: string): void

// 获取倒计时剩余时间（秒）
getCountdownRemaining(id: string): number
```

##### 回调注册
```typescript
// 注册定时器执行回调
onTimerExecute(
  callback: (device: Device, action: 'on' | 'off') => void
): void
```

#### 使用示例

```typescript
import { useDevicesStore } from '@/features/device/stores'

const store = useDevicesStore()

// 添加设备
store.addDevice({
  name: '客厅主灯',
  type: 'light',
  lightType: 'ceiling-lamp',
  location: '客厅',
  status: 'online'
})

// 切换设备状态
store.toggleStatus(deviceId)

// 设置亮度
store.setBrightness(deviceId, 80)

// 设置定时关闭
store.setTimer(deviceId, 'off', true, '22:00', false)

// 设置倒计时
store.setCountdown(deviceId, 'off', 3600)  // 1小时后关闭

// 注册定时器触发回调
store.onTimerExecute((device, action) => {
  console.log(`${device.name} ${action === 'on' ? '已开启' : '已关闭'}`)
})
```

#### 数据持久化

- 使用 `localStorage` 存储设备数据
- 应用启动时自动加载
- 数据变化时自动保存
